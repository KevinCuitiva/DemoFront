# Modulo competition

Responsabilidad:
- Mostrar experiencia competitiva del torneo para participantes.
- Exponer vistas de dashboard, eventos, partidos, calendario y tabla.

Paginas clave:
- `Dashboard.tsx`: vista principal para acciones rapidas del torneo.
- `Events.tsx`: listado y estado de eventos.
- `Matches.tsx`: tarjetas de partidos y resumen por estado.
- `Schedule.tsx`: cronograma por fechas.
- `Scores.tsx`: tabla de posiciones y podio.

Criterios de mantenimiento:
- Mantener consistencia visual entre estados de partido/evento.
- Evitar texto hardcodeado duplicado para etiquetas comunes.
- Validar responsividad en cards, tablas y timelines.
